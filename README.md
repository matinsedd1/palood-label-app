# سیستم چاپ اتیکت فروشگاهی (Store Label App)

یک وب‌اپلیکیشن یکپارچه برای مدیریت، تولید و چاپ اتیکت‌ها و بارکدهای فروشگاهی. این سیستم به صورت مستقیم به Google Sheets متصل می‌شود تا اطلاعات کالاها را فراخوانی کرده و لاگ عملیات چاپ را ذخیره کند.

## ویژگی‌های اصلی
* **اتصال به گوگل شیت:** همگام‌سازی لحظه‌ای داده‌ها و خواندن لیست کالاها.
* **تولید بارکد:** ساخت خودکار بارکد برای محصولات جهت چاپ.
* **ثبت لاگ عملیات:** ثبت تاریخچه تمام اتیکت‌های چاپ‌شده (تاریخ، نام کالا، قیمت و...) در شیت مجزا.
* **پشتیبانی از PWA:** قابلیت نصب برنامه روی گوشی و سیستم‌های دسکتاپ.
* **طراحی واکنش‌گرا (Responsive):** رابط کاربری مدرن و سریع با استفاده از Tailwind CSS.

## تکنولوژی‌های استفاده شده
* **فرانت‌اند:** React.js, Vite, Tailwind CSS
* **بک‌اند:** Node.js, Express
* **پایگاه داده / API:** Google Sheets API (googleapis)

### پیش‌نیازها
* نصب بودن [Node.js](https://nodejs.org/) روی سیستم.
* داشتن اطلاعات احراز هویت گوگل (Client ID و Client Secret) برای اتصال به Google Sheets.
